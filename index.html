PrepTesFiches

1. Présentation

L'application "PrepTesFiches" est une application pour les instituteurs, et prof des écoles.
Elle permet de créer des fiches de préparations et de les gérer. Elle permet aussi de les imprimer.

2. Caractéristiques de l'application.

Logiciels : Android (min v3.0)
Version actuel : 0.3.1 (Alpha-Test)
Sauvegarde : Interne
Permissions nécessaires : Aucune

3. Description

  • fr.bshow.fichedepreparation
    Package principale de l'application : MenuPrincipal : Activité principale. Lancé au démarrage.
    NouvelleFiche : Activité Nouvelle Fiche. Lancé lors de la création de nouvelle fiche
    AncienneFiche : Activité Ancienne Fiche. Lancé lors de la gestion des anciennes fiches
    ChargeEtSauveService : Service de Load/Save. Lancé pendant tout la durée de vie de l'appli
    LayoutManager : Classe LayoutManager. Délivre les écrans en fonction de l'activité appelante, et de l'objet utilisé pour le préremplissage.
  • fr.bshow.fichedepreparation.adapter
    Package contenant les différents adapters pour les ListView et ExpandableListView.
  • fr.bshow.fichedepreparation.constantes
    Package contenant les constantes utilisés dans l'application.
  • fr.bshow.fichedepreparation.enumers
    Package contenant les classes Enumeration.
  • fr.bshow.fichedepreparation.intefac
    Package contenant les Interfaces.
  • fr.bshow.fichedepreparation.lib
    Package contenant les classes Principales de l'application.
